# PAYDAY — 페이데이 (최종본)

## 실행 방법 (아주 간단)
```bash
npm i
npm run dev
```
- 대시보드: http://localhost:3000
- 교육 허브: http://localhost:3000/education

## 핵심 기능
- **버튼 중심** UI: 자산(BTC/ETH), 선형/캔들, 타임프레임(5m/10m/1h/1d/1w/1M/1Y)
- **실시간**: Binance WebSocket + **REST 시드**로 과거 캔들 채우기
- **통화**: USD↔KRW, 환율 자동 갱신(/api/fx)
- **손익 계산**: 자본/레버리지/리스크% + 원클릭(현재가로 엔트리, 스탑-50, 테이크+150), 프리셋(돌파/되돌림/박스)
- **안전장치**: 리스크%>2%, 마진사용률>30% 경고
- **온보딩**: 첫 방문 가이드 모달
- **로컬 저장**: 언어/테마/통화/환율 기억
