
# PAYDAY — 한 번에 실행 (원클릭)

## ⬇️ 어떻게 실행하나요?
### Windows
1) `RUN_WINDOWS.bat` 더블클릭
2) 자동으로 창이 열리고, 브라우저가 `http://localhost:3000`을 엽니다.

### macOS
1) `RUN_MAC.command` 파일을 우클릭 → "열기" (처음 실행 시 권한 확인)
2) 자동으로 터미널이 열리고 브라우저에서 `http://localhost:3000`이 열립니다.

### Linux
1) 터미널에서: `bash RUN_LINUX.sh`
2) 브라우저에서 `http://localhost:3000`

## ❗문제가 생기면
- Node.js가 설치되어 있어야 합니다 → https://nodejs.org/
- 회사망/보안SW가 포트를 막으면 3000 대신 다른 포트가 뜰 수 있습니다(터미널 메시지 확인).
- 아주 오래된 PC에서 처음 `npm i`는 시간이 조금 걸릴 수 있어요.
